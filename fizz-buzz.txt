Pseudocode cara membuat perulangan
1.START
2. DECLARE n, i
3. STORE "n" with any value
4. SET i = 1
5. WHILE i<=n 
6. DO 
7. IF i mod 3 = 0
8. 	DISPLAY "Fizz"
9. ELSE
10. IF i mod 5 = 0
11. 	DISPLAY "Buzz"
12. ELSE 
13. IF i mod 15 = 0
14. 	DISPLAY "FizzBuzz"
15. ELSE 
16. 	DISPLAY = "i"
17. i = i + 1
18. END
